Pj 5. Optical Illusion

REFLECTION

For this project, I tried to create an moire pattern animation. I made the base pattern on illustrator and tried to use svg to p5.js converter to put it in my file and adjust color. However, the conversion did not work well because the object I made in Illustrator was too big. Therefore, unfortunately, I had to insert the image as jpeg file.

Then I created the stripe pattern using moduler(%). Since I had three frames of animation, I made every third stripe to be transparent. After that, I created bouncing animation of the stripe pattern to move back and forth. I also wanted to adjust the speed by pressing the mouse, however, it did not work how I intended. It was not bouncing. So I had to remove it.

I feel like in real life, this could become a nice secret code since the pattern has to be well measured and it is hard to be recognized when the animation is complex.
